for 循环语法

语法结构：

for（循环条件初始化；循环条件判断；循环迭代）{
  循环体；
}

执行规则：

第一步：循环条件的初始化（只在最开始执行一次）
第二步：循环条件判断
1》若判断条件结果为真：则执行循环体；
2》若不成立：结束循环；
第三步：循环迭代
第四步：回到第二步

举例：

1、打印 5 行 5 列的星星

//  打印5行5列的星星

let star = ''
for (var i = 0; i < 5; i++) {
  for (var j = 0; j < 5; j++) {
    star += '*'
  }
  star += '\n'
}
console.log(star)

对象

1.对象的创建

1）字面量创建

let obj = {
  name: '我是一个对象',
  age: 21
}

2）使用 Object 构造器创建对象

let obj = new Object()

3）使用构造函数创建对象

function Obj(name,age){
    this.name = name
    this.age = age
}

// 创建构造函数
const fn = new Obj('小红',21)
console.log(fn);  { name:'小红',age:21 }

2.使用 for…in 遍历对象

let obj = { a: 1, b: 2 }
for (let i in obj) {
  console('key:' + i + ', value:' + obj[i])
}
