jQuery-基本操作

1、入口函数

DOM 中如果在 HTML 结构之前写 JS 代码,需要设置加载事件,获取 HTML 元素

window.onload = function () {
  var div = document.querySelector('div')
  console.log(div)
}

JQ 如果在 HTML 结构之前写 js 代码,需要设置加载事件(入口函数)

$(function () {
  var div = document.querySelector('div')
  console.log(div)
})

2、$符号介绍

1.就是 JQuer 的一个别名

$(function(){})

2.可以如下写法:

jQuery(function () {})

3.是 JQ 中的顶级对象

$(function () {
  console.dir($)
})

3、DOM 对象和 jQuery 对象转化

DOM 对象:

​ 通过 document 方式获取到的元素都叫 DOM 对象

jQuery 对象

​ 通过 $ 方式获取到的元素都叫 JQ 对象

将 JQ 对象转为 DOM 对象方式:

  • JQ 对象[索引]
  • JQ 对象.get(索引)
  • 将 DOM 对象转化为 JQ 对象:
  • $(dom 对象)

4、获取元素

1.语法: $(‘选择器’);

2.选择器可以是 CSS 中的选择器

3.选择器可以是 jQ 中提供的: :first :last :odd (获取索引是奇数的元素) :even (获取索引是偶数的元素) :eq(index)

4.通过方法找对应的元素

5.parent() 或者 parents(选择器)

6.children([选择器])

7.find(选择器)

8.siblings(选择器)

9.nextAll([选择器])

10.prevAll([选择器])

11.eq(值)

5、注册事件

语法:

JQ对象.事件名称(function () {
  处理程序
})

事件:

  • 点击事件: click
  • 鼠标进入事件: mouseenter
  • 鼠标离开事件: mouseleave

6、设置样式

$(‘元素’).css(‘属性’, 值); 设置单个属性样式

$(‘元素’).css({‘属性’: ‘值’,’属性’: ‘值’}) 设置多个属性样式

$(‘元素’).addClass(‘类名 类名’);

$(‘元素’).removeClass(‘类名’);

$(‘元素’).toggleClass(‘类名’);
