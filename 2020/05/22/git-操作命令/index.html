git-操作命令

git init
本地初始化仓库
git add .
将项目提交到缓存区
git commit -m “备注说明”
将项目提交到本地仓库
git branch “新分支名称”
git checkout master
切换主分支
git branch -d 分支名称
删除分支(切换到主分支，并且合并完之后)
git remote add origin SSH 地址
关联远程（与远程建立联系）
git push -u origin master
执行远程同步
多次同步 git push
git clone SSSH 地址
克隆项目
git pull origin master
更新远程代码（下拉）
git log / git log –oneline
查看本地有几次提交记录
git resset –hard 序号
回滚（查看之前提交过的存档文件）
git branch
查看本地所有分支
git remote show origin
查看远程仓储分支
git status (-s)
查看文件状态
git checkout 新分支名
切换新分支
git merge 分支名
合并分支
